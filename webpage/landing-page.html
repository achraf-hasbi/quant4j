What is Quant4J?

Financial calculations that fit naturally inside Java codebases.

Quant4J is an open-source Java library for quantitative finance. It provides a clean, interface-driven API for fixed income analytics, with zero external runtime dependencies, designed for developers who need precise, testable financial calculations embedded in their applications.

Feature highlights

Fixed income building blocks for production systems.

💵

Bond Pricing

Price bonds from market yield or zero curves using clear pricing abstractions.

📉

Duration & Risk

Measure Macaulay duration, modified duration, and DV01 for rate sensitivity workflows.

🎯

Yield to Maturity

Compute YTM through root-finding calculators built for deterministic testing.

🧱

Curve Construction

Bootstrap zero curves from par bonds or market prices for fixed income valuation.

∞

Compounding

Switch between continuous and discrete compounding strategies through explicit APIs.

√

Solvers

Root-finding primitives support yield, calibration, and future analytics modules.

Zero dependencies

Pure Java. No third-party runtime dependencies.

Keep quantitative finance logic portable, auditable, and simple to embed. Quant4J is built to integrate cleanly into services, batch jobs, trading tools, and internal risk applications without dragging a dependency graph behind it.

Quick install

Add Quant4J to your Java project.

Use Maven or Gradle and start with the fixed income APIs.

Maven

<dependency>
  <groupId>io.github.achrafhasbi</groupId>
  <artifactId>quant4j</artifactId>
  <version>1.0.0</version>
</dependency>

Gradle

dependencies {
  implementation("io.github.achrafhasbi:quant4j:1.0.0")
}
