# quant4j (v0.1)

A tiny, Java-native quantitative finance library focusing on **bonds** and **yield curves**.

## Features (v1)
- Discrete & continuous compounding
- Continous and discrete rate conversions
- Discount factors calculation
- Forward rate calculation
- Bond Pricing:
    - by yield (discrete or continuous)
    - by curve (discounting)
- Bootstrapping: build a zero curve from deposits

## Build
```bash
mvn test
```
